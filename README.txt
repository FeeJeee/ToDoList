1. Выполнить скрипт создания базы данных "create_db.sql"
2. Открыть проект и выполнить команду "composer update"
3. Создать файл .env 
4. Выполнить команду "php artisan key:generate"
5. Установить соответсвующие данные для подключения к БД в файле ".env"
6. Для создания таблиц и заполения их тестовыми данными выполнить миграцию "php artisan migrate" 
7. Выполнить команду "php artisan serve"
